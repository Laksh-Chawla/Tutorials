Chapter 1 - Head Element

This is the tag that contains backend data of html file.
Note: This tag is not visible on the webpage.

1. <head> - This tag is used to define the head element of an HTML document.
2. <title> - This tag is used to define the title of the document, which appears in the browser tab.
3. <meta> - This tag is used to define metadata about the document, such as character set,viewport settings, author and description.
4. <link> - This tag is used to link external resources like stylesheets.
5. <style> - This tag is used to define internal CSS styles for the document.

Chapter 2 - Text Basics

Headings

1. <h1> - This tag is used to define the largest heading.
2. <h2> - This tag is used to define the second largest heading.
3. <h3> - This tag is used to define the third largest heading.
4. <h4> - This tag is used to define the fourth largest heading.
5. <h5> - This tag is used to define the fifth largest heading.
6. <h6> - This tag is used to define the smallest heading.

Paragraphs

1. <p> - This tag is used to define a paragraph.
2. <br> - This tag is used to insert a line break.
3. <hr> - This tag is used to insert a horizontal rule (line).

Text Formatting

1. <strong> - This tag is used to define strong text (bold).
2. <em> - This tag is used to define emphasized text (italic).
3. <mark> - This tag is used to highlight text.
4. <small> - This tag is used to define smaller text.
5. <del> - This tag is used to define deleted text (strikethrough).
6. <ins> - This tag is used to define inserted text (underline).
7. <sub> - This tag is used to define subscript text (H2O).
8. <sup> - This tag is used to define superscript text (E = mc2).
9. <code> - This tag is used to define code text (e.g., console.log()).
10. <blockquote> - This tag is used to define a block of quoted text.
11. <cite> - This tag is used to define a citation.
12. <q> - This tag is used to define a short inline quotation.
13. <abbr> - This tag is used to define an abbreviation or acronym.
14. <dfn> - This tag is used to define a term.
15. <kbd> - This tag is used to define keyboard input.
16. <samp> - This tag is used to define sample output from a computer program.
17. <var> - This tag is used to define a variable in programming.
18. <time> - This tag is used to define a specific time or date.
19. <address> - This tag is used to define contact information for the author of a document or article.

Chapter 3 - List Types

Lists cannot be inside Paragraphs

Ordered List

1. <ol> - This tag is used to define an ordered list. Like this:

  1. Item 1
  2. Item 2
  3. Item 3

2. <li> - This tag is used to define a list item within an ordered or unordered list.
3. <li value="2"> - This tag is used to define a list item with a specific value in an ordered list.
4. <ol type="A"> - This tag is used to define an ordered list with uppercase letters (A, B, C, ...).
5. <ol type="a"> - This tag is used to define an ordered list with lowercase letters (a, b, c, ...).
6. <ol type="I"> - This tag is used to define an ordered list with uppercase Roman numerals (I, II, III, ...).
7. <ol type="i"> - This tag is used to define an ordered list with lowercase Roman numerals (i, ii, iii, ...).

Unordered List

1. <ul> - This tag is used to define an unordered list. Like this:

  • Item 1
  • Item 2
  • Item 3

2. <ul type="disc"> - This tag is used to define an unordered list with disc bullets.
3. <ul type="circle"> - This tag is used to define an unordered list with circle bullets.
4. <ul type="square"> - This tag is used to define an unordered list with square bullets.
5. <ul type="none"> - This tag is used to define an unordered list with no bullets.

Definition List

1. <dl> - This tag is used to define a definition list.
2. <dt> - This tag is used to define a term in a definition list.
3. <dd> - This tag is used to define the description of a term in a definition list.

Chapter 4 - Links

Links are used to navigate from one page to another or to different sections within the same page.
Note: Links can also be used to download files, open email clients, or make phone calls.
Tag: <a></a>

1. <a href="URL"> - This tag is used to define a hyperlink.
2. <a target="_blank"> - This attribute is used to open the link in a new tab.
3. <a download> - This attribute is used to download the linked file instead of navigating to it.
4. <a rel="nofollow"> - This attribute is used to tell search engines not to follow the link.
5. <a rel="noopener noreferrer"> - This attribute is used for security reasons when opening links in a new tab.
6. <a name="anchor"> - This attribute is used to create an anchor link within the same page.
7. <a href="#anchor"> - This attribute is used to link to an anchor within the same page.
8. <a href="mailto:> - This attribute is used to create a mailto link that opens the user's email client.
9. <a href="tel:> - This attribute is used to create a link that allows users to call a phone number directly from their device.
10. <a href="javascript:void(0);"> - This attribute is used to create a link that does not navigate anywhere, often used for JavaScript actions.


Note: The <section> tag is used to define a thematic grouping of content, typically with a heading.

Images are used to display pictures or graphics on a webpage.
Tag: <img>
Note: The <img> tag is a self-closing tag, meaning it does not require a closing tag.
